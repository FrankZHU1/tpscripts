What Eclipse plugin that intecrates a Marathon Docker Deployment With Kibana Logs to easily debug Deployed Servers and read their logging output Also contains sql validation for flyway migration files to models
How
  1. Open Eclipse
  2. click on help
  3. click on Install New Software
  4. click on Add
  5. Add the current Url PluginLink as Location (Optionally define an easy to remember name)
  6. Confirm on Add
  7. Choose the Feature Project
  8. Follow the install guide
  9. Open the views
    • click on window
    • Show View
    • click on other
    • goto SocialDear
Additional
  • set Marathon login in Settings (will redirect)
